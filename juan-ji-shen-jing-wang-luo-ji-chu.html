大纲

全连接神经网络

  • Input layer： X

  • hidden layer ：Z

  • Output layer：Y

  • Z = W * X

卷积网络

卷积层

  • Input Volume：输入数据，当输入数据为图像时存在三个通道，即RGB，对应下图中的x[:,:,0]，x[:,:,1]，x[:,:,2] 三个维度的矩阵

  • Filter W0 和 Filter W1：卷积核，卷积核的维度对应输入数据的维度，卷积核的数量决定输出数据的维度

  • Output Volume：输出数据

卷积核

  • 卷积核的数据是通过学习得到的
  • 卷积核的大小（3X3,5X5,7X7等等）是事先给出的

Padding

为了保证图像卷积前后的大小不变，需要添加padding，即为输入数据添加n行或列数值为0的数据

之前在讨论卷积神经网络的时候，我们是使用 filter 来做元素乘法运算来完成卷积运算的。目的是为了完成探测垂直边缘这种特征。但这样做会带来两个问题。

  • 卷积运算后，输出图片尺寸缩小
  • 越是边缘的像素点，对于输出的影响越小，因为卷积运算在移动的时候到边缘就结束了。中间的像素点有可能会参与多次计算，但是边缘像素点可能只参与一次。所以我们的结果可能会丢失边缘信息。

那么为了解决这个问题，我们引入 padding， 什么是 padding 呢，就是我们认为的扩充图片， 在图片外围补充一些像素点，把这些像素点初始化为 0. 如下图：

经过 padding 之后，原始图片尺寸为 (n+2p) x (n+2p)，filter 尺寸为 f x f，则卷积后的图片尺寸为 (n+2p-f+1) x (n+2p-f+1)。若要保证卷积前后图片尺寸不变，则 p 应满足：

卷积核在输入数据上滑动，每次滑动后输入数据与卷积核对应位置相乘再相加所得到的数值再加上偏移量Bias即可得到一个输出数据。

激活函数

  • 卷积网络是非线性模型，主要原因是激活函数的存在。
  • 代码中Conv2D()存在activation参数表示卷积层使用什么激活函数，例如Conv2D('activation='sigmoid')

Sigmoid激活函数

x = np.linspace(-5,5,100)
y = 1 / (1 + np.exp(-x))
plt.plot(x,y,'r-',lw=2,label='Sigmoid')
plt.legend(loc='best',fontsize=14)
plt.show()

Tanh激活函数

Tanh函数 = 2*Sigmoid函数(2*x) - 1

x = np.linspace(-5,5,100)
y = (np.exp(x)-np.exp(-x)) / (np.exp(x) + np.exp(-x))
plt.plot(x,y,'r-',lw=2,label='Tanh')
plt.legend(loc='best',fontsize=14)
plt.show()

ReLU函数

  • ReLU函数
x = np.linspace(-5,5,101)
y = p.copy(x)
y[y<0] = 0
plt.plot(x,y,'r-',lw=2,label='ReLU')
plt.legend(loc='best',fontsize=14)
plt.show()
  • Leaky ReLU函数
x = np.linspace(-5,5,101)
y = p.copy(x)
index = y < 0 
y[index] = 0.05 * y[index]
plt.plot(x,y,'r-',lw=2,label='Leaky ReLU')
plt.legend(loc='best',fontsize=14)
plt.show()

正则化

  • Keras的Conv2D函数中存在kernel_regularizer和bias_regularizer参数，分别对应L1/L2

    例如： Conv2D(kernel_regularizer = 'l1', bias_regularizer = 'l2')

  • Dropout

​ 代码写法例如：model.add(Dropout(0.5))

  • Dropout的方法和特点
    • 对于每一个训练样本，以概率0.5选择隐层的单元是否有效；
    • 几乎每一个输入样本的网络都是不同的，但有效单元的权值相同；
    • 输入层也可以采用dropout（如以0.2的概率drop）；
    • 测试样本下的网络，其实是训练样本得到网络的“平均网络”；
    • 平均网络保证了比单个网络的精度更高；
    • 由于计算中忽略一半的单元，所以计算更加高效；
    • 可以配合预处理得到更好的结果；

池化层Pooling

  • Pooling实际上就是为了降采样

    代码写法例如：model.add(MaxPooling2D((2,2), 2, padding=’valid’))

Pooling层主要的作用是压缩数据和参数的量（保持最显著的特征），通过去掉Feature Map中不重要的样本，进一步减少参数数量。Pooling的方法很多，通常采用最大池化

  • max_pooling:取池化窗口的最大值
  • avg_pooling:取池化窗口的平均值

池化层计算

池化层也有窗口的大小以及移动步长，那么之后的输出大小怎么计算？计算公式同卷积计算公式一样

计算：224x224x64,窗口为2，步长为2输出结果？

​ H2 = (224 - 2 + 2*0)/2 +1 = 112

​ w2 = (224 - 2 + 2*0)/2 +1 = 112

通常池化层采用 2x2大小、步长为2窗口
